   https://your-app-name.streamlit.app

=====================================================================
TROUBLESHOOTING
=====================================================================

PROBLEM: pip install fails
SOLUTION: Update pip first: pip install --upgrade pip

PROBLEM: App won't start
SOLUTION: Check Python version: python --version (need 3.8+)

PROBLEM: "Module not found" error
SOLUTION: Make sure you're in the pardeep_portfolio folder

PROBLEM: PardeepBot gives generic answers
SOLUTION: Add your Gemini API key in secrets.toml

PROBLEM: Can't deploy to Streamlit Cloud
SOLUTION: Make sure repository is public and secrets are added
